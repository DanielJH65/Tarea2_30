Como realizar una clase

Se crea una clase nueva junto al archivo principal main.
El nombre del nuevo archivo sera el nombre de la clase.

Se crean los atributos de la clase.
Estos son las caracteristicas del objeto que se crea a partir de esta clase.
Por buena practica se crean privados para encapsularlos.

Se crea el contructor, esto es lo primero que se ejecuta al momento de instanciar la clase.
Como se ve en la imagen se puede crear el constructor vacio o también se puede crear para que inicialice los valores de los atributos de una vez.

Se crean los métodos Get y Set.
Los Set funcionan para cambiar el valor de un atributo.
Los Get funcionan para obtener el valor de un atributo.

Se crean los métodos de la clase.
Los métodos son acciones que realizará el objeto que se instancia de la clase.

Para instanciar la clase se debe llamar a la clase y asignarsela a un objeto.

Si se usa el constructor vacio para instanciar la clase se deben inicializar los valores de los atributos utilizando los métodos set creados anteriormente.

Se llaman a los métodos que se necesitan utilizar.
En el caso del ejemplo que se ve en la imagen se utiliza los métodos get para obtener los datos que se envian como parametros.
